# 📘 Atividade de Vetores em JavaScript

## 📌 Descrição

Este repositório contém exercícios de lógica de programação utilizando vetores em JavaScript.
Os programas foram desenvolvidos sem utilizar funções prontas para arrays, com o objetivo de praticar manipulação manual de vetores, estruturas de repetição e algoritmos básicos.

---

# 🎯 Objetivos da Atividade

* Desenvolver programas em JavaScript utilizando arquivos `.js`
* Criar algoritmos utilizando vetores
* Praticar estruturas de repetição e condicionais
* Trabalhar manipulação de arrays sem funções prontas
* Testar e validar o funcionamento dos programas
* Aplicar conceitos básicos de lógica de programação
* Utilizar Git e GitHub para versionamento dos arquivos
* Organizar os algoritmos em repositório local e remoto

---

## 🛠️ Ferramentas Utilizadas

* JavaScript
* Visual Studio Code
* Flowgorithm
* Git e GitHub
* Navegador Web / Node.js

# 📋 Exercícios Desenvolvidos

## 1️⃣ Soma dos números pares

Crie um programa que:

* Leia 10 números inteiros
* Armazene os valores em um vetor
* Calcule a soma apenas dos números pares
* Exiba o resultado final

---

## 2️⃣ Vetor com números aleatórios

Crie um programa que:

* Gere automaticamente 7 números aleatórios
* Armazene os valores em um vetor
* Exiba os números gerados

---

## 3️⃣ Contagem de ocorrências

Crie um programa que:

* Solicite 10 números inteiros ao usuário
* Armazene os números em um vetor
* Solicite um novo número
* Verifique quantas vezes esse número aparece no vetor

---

## 4️⃣ Remoção de valores duplicados

Crie um programa que:

* Solicite 10 números ao usuário
* Armazene os valores em um vetor
* Crie um novo vetor sem valores repetidos
* Exiba:

  * Vetor original
  * Vetor sem duplicatas
